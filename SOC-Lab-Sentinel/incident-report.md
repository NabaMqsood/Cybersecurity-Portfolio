# Incident Report – Brute Force Attack

## Summary
Detected 18 failed login attempts within 4 minutes from IP 192.168.1.25 targeting user 'admin'.

## Indicators
- Repeated failed logins
- Same IP address
- Short time interval

## Analysis
The activity indicates a brute-force attack attempting unauthorized access.

## Action Taken
- Flagged as suspicious
- Recommended account lockout policy
- Suggested IP blocking

## Conclusion
Classified as a medium severity security incident.
